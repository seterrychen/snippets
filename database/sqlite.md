## Install
Debian: sqlite3

## Usage
### connect
```
$ sqlite3 database.sqlite
SQLite version 3.22.0 2018-01-22 18:45:57
Enter ".help" for usage hints.
sqlite>
```

### list tables
```
sqlite> .tables
builds      logs        nodes       perms       secrets     steps
cron        migrations  orgsecrets  repos       stages      users
sqlite>
```
